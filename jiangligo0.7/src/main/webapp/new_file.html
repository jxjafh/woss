本管理系统是一个基于WEB网络技术的网站信息收集整理组织、审核平台。

一.系统管理员操作制度

1.1 系统管理员在进行增加会员的过程中，必须进行相关密码及权限的设定.

1.2 管理用户的设定必须经过部门领导的同意，并备案方可进行。

1.3 对于用户权限的设定必须根据用户的具体需求，并根据网站信息发布制度的相关规定执行。

1.4 系统数据的备份需严格遵守关于数据安全管理制度的相关规定进行。

1.5 对于网站的防篡改软件，需实时监控，并按照相关的规定执行。

二.网站审核人员操作制度

2.1 审核人员应根据自己负责的每个栏目对文档及时进行审核并发布。

三.系统维护人员操作制度

3.1 系统维护人员主要负责网站的广告栏目增加、网站编辑用户和网站审核用户、名册的下载。

3.2 分类添加后需报交分类详细说明书，并对网站编辑人员进行培训。

3.3 广告栏目的更改必须经过部门领导批准方可发布。

3.4 文档的下载前需进行审核，并保证及时下载。
